PURPOSE
This package creates the exact approved Sprint 2 replacement file for the
main dexgra-ui/NLDG9 repository.

HOW TO CREATE THE UPLOAD FILE
1. Open CREATE-GAMES-FILE.html in Safari, Chrome, Edge, or Firefox.
2. Press "Create games.html."
3. Your browser will download games.html.

HOW TO UPLOAD TO GITHUB
1. Open the dexgra-ui/NLDG9 repository.
2. Open the repository root.
3. Choose Add file, then Upload files.
4. Upload the newly created games.html.
5. Replace the existing games.html when prompted.
6. Commit the change.
