# Sprint 5.0 — Personalized Platform Experience

## Scope

- Local-device ministry profile
- Favorites and saved content
- Reading and activity history
- Study progress dashboard
- Personal collections
- Personalized content feed
- Installable PWA foundation
- Offline shell and core content access

## Security boundary

NLDG9 is currently hosted as a static GitHub Pages application. Sprint 5.0 does not claim secure server-side authentication or cross-device account syncing. Profile and personalization data remain in the browser through localStorage until a protected backend is introduced.
