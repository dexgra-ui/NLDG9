# Sprint 7.2 — Scripture Study Library

## Theme
Helping believers return to what God has been teaching them.

## Initial scope
- Dedicated searchable study library
- Search by title, passage, topic, audience, and note content
- Filters for study mode and date
- Study cards with passage, themes, last updated date, and progress
- Continue-study handoff back to Interactive Bible Study
- Ministry Tools and My Journey handoffs
- Local-first storage using the existing study notebook data
- Responsive desktop, iPad, and phone layouts

## Ministry boundary
The library organizes user-created notes and curated study metadata. It does not add Scripture text, live AI, divine direction, or authoritative interpretation.
